## 如何使用這份指南

1. **依下方「推薦學習路線」**選擇你的階段，不一定要按章節編號順序
2. **先讀說明**，再打開對應的 `.java` 原始碼對照
3. 每一課**自己手打一遍**程式碼（不要複製貼上！）
4. 每章結束後**完成練習題**，再進入下一章
5. 難度標註：無標記 = 入門、`<!-- 💡 進階 -->` = 中級、`<!-- 🔴 資深 -->` = 高級

## 常見錯誤與解法

| 錯誤訊息 | 原因 | 解法 |
|---------|------|------|
| `ArrayIndexOutOfBoundsException` | 陣列索引超出範圍 | 確認索引 < 陣列長度 |
| `NullPointerException` | 對 null 物件呼叫方法 | 確認物件已初始化，或用 Optional |
| `ClassCastException` | 向下轉型時型別不符 | 先用 instanceof 確認型別 |
| `ArithmeticException` | 整數除以 0 | 加判斷 divisor != 0 |
| `FileNotFoundException` | 檔案路徑不存在 | 確認路徑與工作目錄 |
| `StackOverflowError` | 無限遞迴或遞迴太深 | 確認遞迴有出口條件 |
| `NumberFormatException` | 字串轉數字格式不對 | 確認字串內容是純數字 |
| 浮點數比較錯誤 | `double` 有精度誤差 | 用 `Math.abs(a-b) < 1e-9` 或改用 `BigDecimal` |
