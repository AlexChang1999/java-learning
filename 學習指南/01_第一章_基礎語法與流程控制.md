## 課前準備

學習 Java 前，你需要知道三件事：
1. 每個 Java 程式都住在一個「類別 (class)」裡面
2. 程式從 `main` 方法開始執行
3. 每行敘述句結尾要加分號 `;`

---

## 第 0 課 — HelloWorld：你的第一個程式

```java
package tw.brad.tutor;                    // 這個類別住在哪個「包裹」裡

public class HelloWorld {
    public static void main(String[] args) {  // 程式的入口點，固定格式
        System.out.println("Hello, World");   // 印出一行文字並換行
    }
}
```

**重點說明：**
- `package`：類似資料夾，用來組織類別。可以先不管它
- `public class HelloWorld`：宣告一個公開類別，名字叫 HelloWorld
- `main` 方法：JVM（Java 虛擬機）固定從這裡開始執行
- `System.out.println(...)`：把括號裡的內容印到螢幕，並自動換行
- `System.out.print(...)`：印出但**不**換行

## 第 1 課 — Brad01：整數型別與變數

```java
public class Brad01 {
    public static void main(String[] args) {
        // 整數型別佔用的位元數（決定能存多大的數）
        // byte: 8位元，範圍 -128 ~ 127
        // short: 16位元
        // int: 32位元（最常用）
        // long: 64位元
        byte var1;          // 宣告一個 byte 型別的變數，還沒給值
        byte var2 = 1;      // 宣告並同時給初始值
        int var3;           // 宣告 int 變數
        var1 = 126;         // 給 var1 賦值
        var3 = var1 + var2; // 計算 126 + 1 = 127，存入 var3
        System.out.println(var3); // 印出 127
        byte $_$;           // 變數名稱可以用 $、_ 開頭，但不可用數字開頭
    }
}
```

**變數命名規則：**
- 只能用英文字母、數字、`$`、`_`
- 不可以用數字開頭
- 區分大小寫（`age` 和 `Age` 是不同變數）

**型別速查表：**
| 型別 | 大小 | 範圍 |
|------|------|------|
| `byte` | 1 位元組 | -128 ~ 127 |
| `short` | 2 位元組 | -32768 ~ 32767 |
| `int` | 4 位元組 | 約 ±21 億 |
| `long` | 8 位元組 | 超大整數 |

## 第 2 課 — Brad02：Scanner 輸入與四則運算

```java
import java.util.Scanner;  // 要使用 Scanner，需要先 import

public class Brad02 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);  // 建立一個從鍵盤讀取的掃描器
        
        System.out.print("X = ");      // 提示使用者輸入（不換行）
        int x = scanner.nextInt();     // 讀取一個整數，存入 x
        System.out.print("Y = ");
        int y = scanner.nextInt();     // 讀取第二個整數，存入 y
        System.out.println("-------"); // 印出分隔線
        
        int r1 = x + y;   // 加法
        int r2 = x - y;   // 減法
        int r3 = x * y;   // 乘法
        int r4 = x / y;   // 整數除法（結果去掉小數）
        int r5 = x % y;   // 取餘數（模運算）
        
        // printf 格式化輸出：%d 代表整數的佔位符，\n 是換行符號
        System.out.printf("%d + %d = %d\n", x, y, r1);
        System.out.printf("%d / %d = %d ...... %d\n", x, y, r4, r5);
    }
}
```

**`printf` 常用格式符：**
| 符號 | 用途 |
|------|------|
| `%d` | 整數 |
| `%f` | 浮點數 |
| `%s` | 字串 |
| `%c` | 字元 |
| `\n` | 換行 |
| `\t` | Tab |

## 第 3 課 — Brad03：浮點數、字元、布林

```java
public class Brad03 {
    public static void main(String[] args) {
        // 浮點數（有小數點的數字）
        float var1 = 12;        // float 要在數字後加 f，或直接給整數
        float var2 = 12.0f;     // 加上 f 明確標示是 float
        double var3 = 12.0;     // double 是預設的浮點數型別，精度更高
        
        // 字元型別：用單引號，只能存一個字元
        char c1 = 'a';    // 英文字母
        char c2 = 65;     // 也可以用 ASCII 碼，65 對應 'A'
        char c3 = '資';   // 中文字也可以（Java 用 Unicode，支援各國語言）
        System.out.println(c1); // 印出 a
        System.out.println(c2); // 印出 A
        
        // 布林型別：只有 true 或 false 兩個值
        boolean b1 = true;
        boolean b2 = false;
    }
}
```

**`float` vs `double`：**
- `double` 精度更高，一般計算推薦用 `double`
- `float` 較省記憶體，通常在圖形/遊戲開發中使用

## 第 4 課 — Brad04：if / else 條件判斷

```java
public class Brad04 {
    public static void main(String[] args) {
        boolean a = false;
        if (!a) {           // ! 是「NOT」運算子，!false 等於 true
            System.out.println("OK");   // 條件成立時執行
        } else {
            System.out.println("XX");   // 條件不成立時執行
        }
        // 因為 !false = true，所以印出 "OK"
    }
}
```

**邏輯運算子：**
| 運算子 | 意義 | 範例 |
|--------|------|------|
| `!` | NOT（取反） | `!true` → `false` |
| `&&` | AND（且） | `true && false` → `false` |
| `\|\|` | OR（或） | `true \|\| false` → `true` |

## 第 5 課 — Brad05：亂數與多層 if-else

```java
public class Brad05 {
    public static void main(String[] args) {
        // Math.random() 產生 0.0 ~ 0.9999... 之間的小數
        double temp = Math.random();
        
        // 轉換成 0 ~ 100 的整數
        // (int) 是「強制型別轉換」，會直接去掉小數部分（不四捨五入）
        int score = (int)(temp * 101);  // 0 ~ 100
        System.out.println(score);
        
        // 多層條件判斷：從最高分往下比
        if (score >= 90) {
            System.out.println("A");
        } else if (score >= 80) {
            System.out.println("B");
        } else if (score >= 70) {
            System.out.println("C");
        } else if (score >= 60) {
            System.out.println("D");
        } else {
            System.out.println("E");
        }
    }
}
```

**亂數公式：**
- 產生 `0 ~ n` 的整數：`(int)(Math.random() * (n+1))`
- 產生 `1 ~ 6` 的整數：`(int)(Math.random() * 6) + 1`

## 第 6 課 — Brad06：特殊數值與進位表示

```java
public class Brad06 {
    public static void main(String[] args) {
        double a = 0;
        int b = 0;
        System.out.println(a / b);  // double 除以 0 = Infinity（無限大），不會報錯
        // 注意：int 除以 0 會拋出 ArithmeticException 錯誤！
        
        int c1 = 10;    // 一般十進位：10
        int c2 = 010;   // 開頭加 0 = 八進位！010 換算成十進位是 8
        int c3 = 0x10;  // 開頭加 0x = 十六進位！0x10 換算成十進位是 16
        System.out.println(c1); // 10
        System.out.println(c2); // 8
        System.out.println(c3); // 16
    }
}
```

**⚠️ 注意：** 整數前面加上 `0` 會被當成**八進位**，這是常見的 bug 來源！

## 第 7 課 — Brad07：switch / case

```java
public class Brad07 {
    public static void main(String[] args) {
        int a = 10;
        switch(a) {           // 括號裡放要比較的值
            case 1:  System.out.println("A"); break;   // 等於 1 時執行
            case 10: System.out.println("B"); break;   // 等於 10 時執行 → 印出 B
            case 100: System.out.println("C"); break;
            default: System.out.println("XX");         // 都不符合時執行
        }
    }
}
```

**⚠️ 記得加 `break`！**
- 如果沒有 `break`，程式會繼續往下執行（稱為「fall-through」）
- `default` 是所有 case 都不符合時的預設處理，類似 else

## 第 8 課 — Brad08：for 迴圈與靜態方法

```java
public class Brad08 {
    public static void main(String[] args) {
        int i = 0;
        // for 迴圈格式：for (初始化; 條件; 更新) { 迴圈體 }
        // 初始化和更新不一定要是賦值，可以是方法呼叫
        for (printBrad(); i < 10; printLine()) {
            System.out.println(i++);  // 先印 i，再 i+1
        }
    }
    
    // static 方法：不需要建立物件，直接用類別名稱呼叫
    static void printBrad() {
        System.out.println("Brad");     // 迴圈開始前執行一次
    }
    
    static void printLine() {
        System.out.println("-------"); // 每次迴圈結束後執行
    }
}
```

**for 迴圈三個部分：**
1. `初始化`：迴圈開始前執行一次
2. `條件`：每次執行前檢查，若為 false 就結束
3. `更新`：每次迴圈體執行完後執行

## 第 9 課 — Brad09：巢狀 if 與閏年判斷

```java
public class Brad09 {
    public static void main(String[] args) {
        int year = 2000;
        boolean isLeap;
        
        // 方法一：巢狀 if（層層判斷）
        if (year % 4 == 0) {
            if (year % 100 == 0) {
                if (year % 400 == 0) {
                    isLeap = true;   // 能被 400 整除 → 閏年
                } else {
                    isLeap = false;  // 能被 100 但不能被 400 整除 → 平年
                }
            } else {
                isLeap = true;  // 能被 4 但不能被 100 整除 → 閏年
            }
        } else {
            isLeap = false;  // 不能被 4 整除 → 平年
        }
        
        // 方法二：合併條件（更簡潔）
        if (year % 400 == 0 || (year % 4 == 0 && year % 100 != 0)) {
            isLeap = true;
        } else {
            isLeap = false;
        }
        
        // 三元運算子：條件 ? 成立時的值 : 不成立時的值
        System.out.printf("%d年是%s年", year, isLeap ? "潤" : "平");
    }
}
```

**閏年規則：** 能被 400 整除，OR（能被 4 整除 AND 不能被 100 整除）

## 第 10 課 — Brad10：& 和 && 的差異

```java
public class Brad10 {
    public static void main(String[] args) {
        int a = 10, b = 3;
        
        // & ：非短路 AND，兩側都會執行
        // && ：短路 AND，左側為 false 就不看右側
        if (++a <= 10 & --b >= 3) {
            // ++a 先執行（a 變 11），--b 也先執行（b 變 2）
            // 11 <= 10 是 false，2 >= 3 也是 false
            // & 兩側都執行，所以 a=11, b=2
            System.out.printf("OK: a = %d, b = %d", a, b);
        } else {
            System.out.printf("XX: a = %d, b = %d", a, b);
            // 印出：XX: a = 11, b = 2
        }
    }
}
```

**`&` vs `&&` 的重要區別：**
- `&&`（短路）：左側為 `false`，右側**不執行** → 效率更高，更常用
- `&`（非短路）：不管左側結果，右側**一定執行** → 適合需要確保兩側都執行的情況

## 第 11 課 — Brad11：巢狀迴圈（九九乘法表）

```java
public class Brad11 {
    public static void main(String[] args) {
        // final 宣告常數（值不可改變）
        final int ROWS = 1;
        final int COLS = 9;
        final int START = 1;
        
        // 三層巢狀迴圈
        for (int k = 0; k < ROWS; k++) {       // 最外層：控制組數
            for (int j = 1; j <= 9; j++) {      // 中層：乘數 1~9
                for (int i = START; i < START + COLS; i++) {  // 內層：被乘數
                    int newi = i + k * COLS;
                    int r = newi * j;
                    System.out.printf("%d x %d = %d\t", newi, j, r);
                }
                System.out.println();  // 每一組乘數印完換行
            }
            System.out.println("-----");
        }
    }
}
```

## 第 12 課 — Brad12：while 迴圈

```java
public class Brad12 {
    public static void main(String[] args) {
        int a = 0;
        
        // while 迴圈：先檢查條件，再執行
        while (a < 10) {
            System.out.println(a++);  // 印出 0~9
        }
        
        // for 迴圈也可以寫成類似 while 的形式（省略初始化和更新）
        // 因為 a 已經是 10，條件不成立，這段不會執行
        for (; a < 10;) {
            System.out.println(a++);
        }
    }
}
```

**三種迴圈比較：**
| 迴圈 | 特點 | 適用場景 |
|------|------|---------|
| `for` | 結構清晰，次數固定 | 知道要跑幾次 |
| `while` | 先判斷後執行 | 不知道次數，有條件控制 |
| `do-while` | 先執行後判斷（至少跑一次）| 需要至少執行一次 |

## 第 13 課 — Brad13：陣列

```java
public class Brad13 {
    public static void main(String[] args) {
        int[] a;        // 宣告一個整數陣列（還沒分配空間）
        a = new int[3]; // 分配空間，可以存 3 個整數，索引 0~2
        
        a[0] = 123;     // 設定第 1 個元素（索引從 0 開始！）
        a[2] = 17;
        a[1] = 333;
        
        // 傳統 for 迴圈遍歷
        for (int i = 0; i < a.length; i++) {  // a.length 是陣列長度
            System.out.println(a[i]);
        }
        
        // a[100]  ← 這行會報 ArrayIndexOutOfBoundsException 錯誤！
        
        System.out.println("---");
        
        // for-each 迴圈（增強型 for）：更簡潔，無法修改元素
        for (int vv : a) {
            System.out.println(vv);
        }
    }
}
```

**陣列重點：**
- 索引從 **0** 開始，最後一個索引是 `長度-1`
- 超出範圍會拋出 `ArrayIndexOutOfBoundsException`
- `int[] nums = {1, 2, 3}` 是快速初始化語法

## 第 14 ~ 16 課 — Brad14/15/16：陣列統計（骰子模擬）

```java
// Brad14：用 switch 統計各點數（6個變數）
public class Brad14 {
    public static void main(String[] args) {
        // 用 6 個獨立變數統計
        int p0, p1, p2, p3, p4, p5, p6;
        p0 = p1 = p2 = p3 = p4 = p5 = p6 = 0;  // 連鎖賦值
        
        for (int i = 0; i < 100; i++) {
            int point = (int)(Math.random() * 6) + 1;  // 1~6
            switch (point) {
                case 1: p1++; break;
                case 2: p2++; break;
                // ...
            }
        }
    }
}

// Brad15：用陣列統計（更優雅）
public class Brad15 {
    public static void main(String[] args) {
        int[] p = new int[7];  // 開7格，索引0不用，1~6對應點數
        
        for (int i = 0; i < 10000000; i++) {  // 模擬一千萬次
            int point = (int)(Math.random() * 6) + 1;
            p[point]++;  // 直接用點數當索引！
        }
        
        for (int i = 1; i <= 6; i++) {
            System.out.printf("%d點出現%d次\n", i, p[i]);
        }
    }
}

// Brad16：用三元運算子做索引轉換
public class Brad16 {
    public static void main(String[] args) {
        int[] p = new int[7];
        
        for (int i = 0; i < 10000000; i++) {
            int point = (int)(Math.random() * 9) + 1;  // 故意產生 1~9
            // point <= 6 就直接用，> 6 就減 3（模擬不均勻骰子）
            p[point <= 6 ? point : point - 3]++;
        }
    }
}
```

## 第 18 課 — Brad18：do-while 迴圈

```java
public class Brad18 {
    public static void main(String[] args) {
        int a = 10;
        
        // while：條件一開始不成立，不會執行
        while (a < 10) {
            System.out.println(a++);  // 這行不會被執行
        }
        System.out.println("---");
        
        // do-while：先執行一次，再判斷條件
        a = 10;
        do {
            a++;
            System.out.println(a);  // 至少執行一次，印出 11
        } while (a < 10);   // 11 < 10 為 false，結束
    }
}
```

## 第 19 課 — Brad19：位元運算 XOR 交換

```java
public class Brad19 {
    public static void main(String[] args) {
        int a = -10, b = 3;
        
        // XOR 交換法：不需要暫存變數
        a = a ^ b;  // a 現在存的是 a XOR b
        b = a ^ b;  // b = (a XOR b) XOR b = a（原來的 a）
        a = a ^ b;  // a = (a XOR b) XOR a（原來的 a）= b（原來的 b）
        
        System.out.printf("a = %d, b = %d\n", a, b);  // a = 3, b = -10
        System.out.println(3 ^ 2);  // 3 XOR 2 = 1（位元運算）
    }
}
```

**位元運算子：**
| 運算子 | 名稱 | 說明 |
|--------|------|------|
| `&` | AND | 兩者都是1才是1 |
| `\|` | OR | 任一是1就是1 |
| `^` | XOR | 不同才是1 |
| `~` | NOT | 取反 |
| `<<` | 左移 | 相當於乘以2的次方 |
| `>>` | 右移 | 相當於除以2的次方 |
