## 專案一：撲克牌洗牌（PokerV1 ~ PokerV4）

這四個版本展示了如何用**逐步優化**的思維改進同一個問題。

### PokerV1：暴力法（每次亂數，重複就重試）

```java
public class PokerV1 {
    public static void main(String[] args) {
        int nums = 52;
        int[] poker = new int[nums];
        
        for (int i = 0; i < poker.length; i++) {
            int temp = (int)(Math.random() * nums);  // 0~51 亂數
            
            // 檢查是否重複
            boolean isRepeat = false;
            for (int j = 0; j < i; j++) {
                if (temp == poker[j]) {
                    isRepeat = true;
                    break;
                }
            }
            
            if (!isRepeat) {
                poker[i] = temp;
            } else {
                i--;  // 重複了就退回去重試
            }
        }
        
        for (int card : poker) System.out.println(card);
    }
}
```

**問題：** 當已填入的牌越來越多，重複機率越來越高，效率越來越差。

---

### 概念解說：Fisher-Yates 洗牌演算法

**正確洗牌的思路：**
1. 先建立一個有序陣列：[0, 1, 2, ..., 51]
2. 從最後一個元素開始，隨機挑一個位置和它交換
3. 重複直到第一個元素

```
初始：[0, 1, 2, 3, 4]
第4步：隨機選 2，交換 poker[4] 和 poker[2] → [0, 1, 4, 3, 2]
第3步：隨機選 1，交換 poker[3] 和 poker[1] → [0, 3, 4, 1, 2]
...
```

這種方法每個排列出現的機率完全相等，而且只需要 O(n) 的時間。

## 專案二：猜數字遊戲（GuessNumber.java）

這是一個完整的 Swing GUI 應用，綜合了第二章學到的所有 OOP 概念。

### 遊戲規則
- 系統隨機產生一個3位不重複數字的謎底（如 "437"）
- 玩家猜一個3位數
- 系統回應 `xAyB`：A = 數字對且位置對，B = 數字對但位置不對
- 最多猜3次

### 核心功能解析

```java
// 1. 產生不重複的謎底
private static String createAnswer(int d) {
    int[] poker = new int[10];
    for (int i = 0; i < poker.length; i++) poker[i] = i;  // [0,1,2,...,9]
    
    // Fisher-Yates 洗牌
    for (int i = 9; i > 0; i--) {
        int r = (int)(Math.random() * (i + 1));
        int temp = poker[i];
        poker[i] = poker[r];
        poker[r] = temp;
    }
    
    StringBuilder sb = new StringBuilder();
    for (int i = 0; i < d; i++) sb.append(poker[i]);  // 取前 d 個
    return sb.toString();
}

// 2. 計算 xAyB
private String checkAB(String g) {
    int a = 0, b = 0;
    for (int i = 0; i < answer.length(); i++) {
        if (g.charAt(i) == answer.charAt(i)) {
            a++;  // 位置和數字都對
        } else if (answer.indexOf(g.charAt(i)) != -1) {
            b++;  // 數字對但位置不對
        }
    }
    return String.format("%dA%dB", a, b);
}

// 3. 事件處理（點擊「猜」按鈕）
@Override
public void actionPerformed(ActionEvent e) {
    String g = input.getText();
    counter++;
    
    String result = checkAB(g);
    log.append(String.format("%d. %s => %s\n", counter, g, result));
    
    if (result.equals("3A0B")) {
        JOptionPane.showMessageDialog(null, "恭喜老爺");
        initGame();  // 重新開始
    } else if (counter == 3) {
        JOptionPane.showMessageDialog(null, "魯蛇: 謎底:" + answer);
        initGame();
    }
}
```

| 概念 | 在哪裡出現 |
